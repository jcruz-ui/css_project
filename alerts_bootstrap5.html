Alerts are useful for communicating things when they go wrong.
E.g. wrong passwords, service maintenance, or missing information.
üö® This is an alert
Let's look at the code:
<div class="alert alert-danger" role="alert">
   This is an alert.
</div> hello
The class keyword, that is class="alert alert-danger", defines the type of alert.
The role keyword, that is role="alert", defines the purpose of the alert.

There are many more types of alerts from which to choose from.
In essence, only the class keyword needs to change. So, instead of writing
class="alert alert-danger" you would write class="alert alert-success".
