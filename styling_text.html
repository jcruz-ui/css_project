How to Style text  on a Web Page using CSS

Make sure to use the proper CDN if you want to try out the example.

Applying style to text makes the regular stand out more. Here we will focus on color. Let's look at the example text below:

      This is a sentence.

There's nothing about it that makes it stand out from the rest of the text. Using the CSS color property, we can change this.

p { color: green; }

Notice that the very first character is used to select the <p> element by simply writng a p before the first curly brace.
