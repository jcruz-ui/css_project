How to use the font-family CSS property

The font-family CSS property specifies the typeface that is used to render the text. The value of this property is a list of comma-separated font family names. The order in the list indicates the priority of the fonts. The browser uses the first font family in the list that is available.

Example code

<p style="font-family: 'Water Brush', cursive;">
   styled text
</p>
